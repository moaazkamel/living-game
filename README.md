It is a game where the player is on an island in the middle of the water and can move around in it.
Challenging platforms

Simple and clean 2D pixel design

Controls: Move Left: Arrow Left Move and A. Right: Arrow Right and D. Move Back: Arrow Down Move or S. Move Front: Arrow Up move or W

Built with: Godot Engine (2D Game Engine)
